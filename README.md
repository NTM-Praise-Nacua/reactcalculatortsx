# React Calculator with Typescript

This version of react calculator applies typescript and uses tailwind v4.

## Setup

1. Clone repo to your selected directory
```bash
git clone https://github.com/NTM-Praise-Nacua/reactcalculatortsx.git
```
2. `cd` to your directory and install packages
```bash
npm i
```
3. Run system
```bash
npm run dev
```
